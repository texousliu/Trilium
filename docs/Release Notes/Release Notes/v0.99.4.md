    *   **‚ÄúMax content width‚Äù option and content alignment improvements** ‚Äì the internal behavior of the ‚ÄúMax content width‚Äù option has been rewritten.     *   **Read-only note alert bar** ‚Äì to better highlight that a note is read-only and make editing more intuitive, a message bar now appears at the top of the note with a button to enable editing.     *   **Quick Edit dialog background tinting** ‚Äì the background tint of the Quick Edit dialog now reflects the note‚Äôs color attribute.
