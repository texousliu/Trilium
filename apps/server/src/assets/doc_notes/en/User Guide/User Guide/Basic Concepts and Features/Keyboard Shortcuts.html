This is supposed to be a complete list of keyboard shortcuts. Note that some of these may work only in certain contexts (e.g. in tree pane or note editor).

It is also possible to configure most keyboard shortcuts in Options -> Keyboard shortcuts. Using global: prefix, you can assign a shortcut which will work even without Trilium being in focus (requires app restart to take effect).

Note navigation

  • Alt + ‚Üê, Alt + ‚Üí ‚Äì go back / forwards in the history
  • Ctrl + J ‚Äì show "Jump to" dialog
  • Ctrl + . ‚Äì scroll to current note (useful when you scroll away from your note or your focus is currently in the editor)
  • Backspace ‚Äì jumps to parent note
  • Alt + C ‚Äì collapse whole note tree
  • Alt + - (alt with minus sign) ‚Äì collapse subtree (if some subtree takes too much space on tree pane you can collapse it)
  • you can define a label #keyboardShortcut with e.g. value Ctrl + I . Pressing this keyboard combination will then bring you to the note on which it is defined. Note that Trilium must be reloaded/restarted (Ctrl + R ) for changes to be in effect.

See demo of some of these features in note navigation.

Only in desktop (electron build):

  • Ctrl + T ‚Äì opens empty tab
  • Ctrl + W ‚Äì closes active tab

Creating notes

  • CTRL+O ‚Äì creates new note after the current note
  • CTRL+P ‚Äì creates new sub-note into current note
  • F2 ‚Äì edit Branch prefix of current note clone

Editing notes

For keyboard shortcuts specific to Text notes, refer to Keyboard shortcuts and  Markdown-like formatting.

  • Enter in tree pane switches from tree pane into note title. Enter from note title switches focus to text editor. Ctrl+. switches back from editor to tree pane.
  • Ctrl+. ‚Äì jump away from the editor to tree pane and scroll to current note

Runtime shortcuts

These are hooked in Electron to be similar to native browser keyboard shortcuts.

  • F5, Ctrl+R ‚Äì reloads Trilium front-end
  • Ctrl+Shift+I ‚Äì show developer tools
  • Ctrl+F ‚Äì show search dialog
  • Ctrl+- ‚Äì zoom out
  • Ctrl+= ‚Äì zoom in
